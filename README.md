# readme

simple skillboost html->pdf page downloader, so you can have those locally and find what you need quicker...

## setup

```sh
make setup
```

## demo

```sh
make demo
```

## batch usage

> write a for loop :)
or use `looper.sh`:

```sh
bash looper.sh paths 16 30
bash looper.sh course_templates 621 640
```
